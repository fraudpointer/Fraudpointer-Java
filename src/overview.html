Java Proxy Client for Fraudpointer Service (ver 1.0)

{@link com.fraudpointer.api.ClientFactory ClientFactory} is the class you need when you start using FraudPointer API. It has a static method that allows you to instantiate a client object. This is {@link com.fraudpointer.api.ClientFactory#construct construct()}.

Sample Code: {@link com.fraudpointer.ConsoleClientExample Click here} if you want to see a sample piece of code that uses the methods of the API.

Developer’s Guide: You can visit Fraudpointer Documentation and read the Developer’s Guide, with extensive details on how you can integrate Fraudpointer Service with your application. (You need to have a valid Fraudpointer Account in order to access the documentation site. If you want to start, visit Account Creation page, fill in your details and follow the procedure to generate an Account.)

Device fingerprinting: Do not forget to take the necessary actions for Device fingerprinting. Click here to read what you have to do.

System Attributes: System Attributes are there for you to start working with Fraudpointer as quickly as possible. You can read about them in the Developer’s Guide, by clicking here.
